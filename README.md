# 🚀 AI Course Marketplace

A production-ready Full Stack AI-powered Course Marketplace built with Next.js, Express.js, MongoDB, Better Auth, JWT Authentication, and Groq LLM.

The platform allows users to explore AI courses, publish their own courses, generate AI-powered course descriptions, and receive personalized AI learning roadmaps based on their goals.
